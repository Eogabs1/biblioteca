# 📚 Catálogo de Livros (Java + Gutendex API)

Este é um aplicativo Java simples que permite buscar livros através da [API do Gutendex](https://gutendex.com/), com uma interface interativa via console.

## 🚀 Funcionalidades

- 🔎 Buscar livros por **título**
- 📝 Listar os **livros mais populares**
- 👤 Buscar livros por **autor**
- 📖 Ver **detalhes de um livro** usando seu ID
- ❌ Encerrar o programa

---

## 🛠️ Tecnologias Usadas

- Java (8+)
- `HttpURLConnection` para chamadas HTTP
- [org.json](https://github.com/stleary/JSON-java) para manipulação de JSON
